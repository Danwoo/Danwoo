# 👋 안녕하세요, Danwoo입니다  

코드로 호기심을 쫓아가며, ontology 기반 RAG부터 AI 에이전트 기술까지 다양한 영역을 탐구하는 개발자입니다. 별이 쌓이지 않아도, 매 커밋마다 성장하는 모습을 보여드리고 싶어요.  

## 🚀 내가 만든 것들  
- **ontology-scaled-rag-finder** – 금융 QA에 ontology‑backed Graph RAG가 언제 도움이 되는지, Vector·Graph·Hybrid 접근을 FIBO 모듈 수를 변수로 실험한 프로젝트. SEOCHO SDK 위에 구현했습니다.  
- **seocho** – 온톨로지와 지식 그래프를 결합해 궁극의 GraphRAG 시스템을 만들기 위한 기초 라이브러리. Python으로 구현했으며, 향후 확장 가능한 모듈 구조를 목표로 합니다.  
- **career-ops** – Claude Code를 backbone으로 한 AI 기반 취업 지원 시스템. 14가지 스킬 모드, Go 대시보드, PDF 생성, 배치 처리 기능까지 갖춰서 구직 과정을 자동화해봤습니다.  
- **agent-skills** – AI 코딩 에이전트가 실제 프로덕션에서 쓸 수 있는 엔지니어링 스킬 집합을 정리한 레포. 베스트 프랙티스와 템플릿을 공유합니다.  
- **langgraph-v1-tutorial** – LangGraph V1을 한글로 쉽게 배울 수 있도록 만든 튜토리얼 시리즈. 초보자도 따라 할 수 있도록 예제 위주로 구성했습니다.  

## 💻 기술 스택  
주로 **Python**(7개 레포)와 **Jupyter Notebook**(4개)를 활용해 데이터 분석과 프로토타입을 만들고, **JavaScript**(1개)로 인터랙티브 대시보드, **Java**(1개)로 백엔드 실험을 해봤습니다. 그 외에도 **Rust**(재작성 중인 claw-code)와 **Go**(career-ops 대시보드) 등 새로운 언어를 constantly 시도하며 영역을 넓혀가고 있습니다.
