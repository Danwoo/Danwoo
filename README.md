# 👋 안녕하세요, Danwoo입니다  

코드로 궁금증을 풀고, 구조화된 지식을 활용해 실‑world 문제를 해결하는 것을 즐깁니다.  
19개의 레포지토리는 저의 호기심과 실험 정신을 담은 작은 실험실이라고 할 수 있어요.  

## 🚀 내가 만든 것들  
- **ontology-scaled-rag-finder** – 온톨로지 기반 Graph RAG가 금융 QA에 언제 도움이 되는지 벡터·그래프·하이브리드 방식을 비교하는 실험 (Python, SEOCHO SDK)  
- **seocho** – 온톨로지와 지식 그래프를 결합해 궁극의 GraphRAG 시스템을 만들어 보는 프로젝트 (Python)  
- **career-ops** – Claude Code를 backbone으로 한 AI 구직 지원 시스템, 14가지 스킬 모드, Go 대시보드, PDF 생성, 배치 처리 (JavaScript)  
- **agent-skills** – AI 코딩 에이전트가 갖추면 좋은 생산성 높은 엔지니어링 스킬 모음 (언어 미지정)  
- **langgraph-v1-tutorial** – LangGraph V1을 한국어로 쉽게 배울 수 있는 튜토리얼 (Jupyter Notebook)  
- **memorial** – 개인적인 기록과 회고를 담은 공간 (Python)  
- **claw-code** – “역사상 가장 빠르게 50K 별을 받은 레포”라는 유머러스한 슬로건 아래, Rust로 재작성 중인 실험 프로젝트  

## 💻 기술 스택  
주로 **Python**(7개)과 **Jupyter Notebook**(4개)로 프로토타입을 만들고, occasional에 **JavaScript**(1)와 **Java**(1)로 웹 컴포넌트를 섞어 사용합니다.  
새로운 그래프·온톨로지 기술과 LLM 에이전트 프레임워크를 탐구하며, 필요에 따라 Rust나 Go도 손대고 있어요.
