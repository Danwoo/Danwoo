## 🚀 내가 만든 것들  

- **[ontology-scaled-rag-finder](https://github.com/Danwoo/ontology-scaled-rag-finder)** – 금융 QA에서 온톨로지 크기별 Graph RAG 성능을 비교하는 실험 프레임워크. SEOCHO SDK 위에 두 트랙 LLM-as-judge 분석을 구현했습니다.

- **[career-ops](https://github.com/Danwoo/career-ops)** – Claude Code를 기반으로 한 AI 취업 지원 시스템. 14가지 스킬 모드, Go 대시보드, PDF 생성, 배치 처리 기능을 갖추고 있어 구직 과정을 자동화합니다.

- **[langgraph-v1-tutorial](https://github.com/Danwoo/langgraph-v1-tutorial)** – LangGraph V1을 한글로 쉽게 배울 수 있도록 만든 튜토리얼 레포지토리. 초보자도 바로 따라 할 수 있도록 단계별 예제를 포함했습니다.
