# 👋 안녕하세요, Danwoo입니다.  
코드로 세상의 작은 불편함을 하나씩 해결해가는 것이 저의 개발 철학입니다. 19개의 레포지토리는 저의 호기심과 탐구 정신의 흔적이며, 각각의 프로젝트는 새로운 기술을 깊이 파고드는 과정의 결과물입니다.  

## 🚀 내가 만든 것들  
- **ontology-scaled-rag-finder** – 금융 QA에 온톨로지 기반 Graph RAG가 언제 도움이 되는지 분석한 실험 프로젝트. Python과 SEOCHO SDK로 구현.  
- **seocho** – 온톨로지와 지식 그래프를 결합해 궁극적 GraphRAG 시스템을 지향하는 핵심 라이브러리.  
- **career-ops** – Claude Code를 기반으로 한 AI 채용 지원 도구. 14가지 스킬 모드, Go 대시보드, PDF 생성 및 배치 처리 기능을 제공.  
- **agent-skills** – AI 코딩 에이전트가 실무에서 발휘할 수 있는 엔지니어링 스킬 모음집.  
- **langgraph-v1-tutorial** – LangGraph V1을 한글로 쉽게 배울 수 있도록 만든 튜토리얼 자료.  

## 💻 기술 스택  
주로 **Python**(7개)과 **Jupyter Notebook**(4개)를 활용해 데이터 및 AI 관련 실험을 진행하며, occasional에 **JavaScript**와 **Java**로 웹 및 백엔드 프로토타입을 만들어 봅니다. 그 외에도 새로운 프레임워크와 언어를 constantly 탐구 중입니다.
